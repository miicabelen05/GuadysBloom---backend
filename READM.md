# GuadysBloom Backend

Proyecto de Backend desarrollado con Node.js, Express, Handlebars y Socket.io.

## Funcionalidades

- API REST de productos
- CRUD de productos
- Query params (?limit)
- Params por ID
- Persistencia con JSON
- Handlebars como motor de plantillas
- Vista en tiempo real con Socket.io

## Rutas principales

/api/products
/api/products/:pid
/realtimeproducts

## Instalación

npm install

## Ejecutar proyecto

npm start
